# HogWords

해리포터 테마 영어 단어 학습 RPG PC 웹 앱.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 을 열면 전체 화면 PC 레이아웃으로 동작합니다.

## 기능

- 스플래시 → 메인 → 단어 학습 → 완료 화면
- 복습 모드, XP/레벨/칭호, 업적, 연속 학습 일수
- Web Speech API 발음(TTS)
- localStorage 저장
- 관리자: 비밀번호 `lumos` (메인 ⚙️ 또는 설정 탭)

## 스택

React, Vite, TypeScript, Tailwind CSS v4, Framer Motion, Zustand, React Router, Lucide React
